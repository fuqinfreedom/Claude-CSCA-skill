# 网络安全与AI咨询工作空间

本工作空间用于产出专业网络安全与AI咨询交付物。所有输出应符合咨询公司交付标准。

## 工作模式

1. **咨询优先**：先理解、分析、规划，再生成内容。不跳过分析直接输出。
2. **四阶段交付**：任何面向客户的交付物必须经过 分析→规划→生成→审查 四个阶段。
3. **两轮审查**：第一轮关键点事实核查 + 第二轮整体质量7维度评分。不达标自动重写。
4. **去AI化**：120+禁用词汇 + 表格去AI规则 + 句式多样性校验。输出读起来像资深顾问写的。
5. **可直接交付**：输出就是最终交付件，不需要二次加工。

## 能力域

### 网安咨询
- **安全评估**：渗透测试 / 代码审计 / 日志分析 / 数据安全 / 应急响应
- **咨询规划**：需求分析 / 建设方案 / 成熟度评估 / 等保咨询
- **培训赋能**：课程设计 / PPT课件 / 实验环境 / CTF题目
- **研发支持**：Java/Python/Go安全Demo / 安全工具开发 / 漏洞PoC
- **商务支撑**：报价 / 人天计算 / 预算编制 / 招投标文件
- **专业写作**：咨询报告 / 制度文档 / 技术方案 / 合规审计

### AI咨询（双方向）
- **AI赋能网安**：AI驱动威胁检测 / AI安全运营(SOC Copilot) / AI代码审计 / AI渗透测试 / AI威胁情报
- **安全赋能AI**：AI安全评估(OWASP LLM Top 10) / 大模型红队测试 / AI合规(算法备案/TC260-003)
- **AI战略**：AI成熟度评估 / AI路线图 / 模型选型 / RAG与Agent架构设计 / Make vs Buy
