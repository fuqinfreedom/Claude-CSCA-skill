# CSCA — 网络安全与AI咨询专家

CSCA（Cyber Security & AI Consulting Agent）是一个运行在 Claude Code 上的专业咨询 Skill，面向网络安全与AI咨询场景，产出可直接交付客户的专业文档。

## 一句话说明

把 Claude Code 变成一个咨询公司的资深顾问团队——自动执行「分析→规划→生成→审查」四阶段交付流程，输出经过去AI化处理的、可直接交付客户的专业咨询文档。

## 核心能力

### 网安咨询
- **安全评估**：渗透测试 / 代码审计 / 日志分析 / 数据安全 / 应急响应
- **咨询规划**：需求分析 / 建设方案 / 成熟度评估 / 等保咨询
- **培训赋能**：课程设计 / PPT课件 / 实验环境 / CTF题目
- **研发支持**：Java/Python/Go安全Demo / 安全工具开发 / 漏洞PoC
- **商务支撑**：报价 / 人天计算 / 预算编制 / 招投标文件
- **专业写作**：咨询报告 / 制度文档 / 技术方案 / 合规审计

### AI咨询（双方向）
- **AI赋能网安**：AI驱动威胁检测 / AI安全运营（SOC Copilot）/ AI代码审计 / AI渗透测试 / AI威胁情报
- **安全赋能AI**：AI安全评估（OWASP LLM Top 10）/ 大模型红队测试 / AI合规（算法备案/TC260-003）
- **AI战略**：AI成熟度评估 / AI路线图 / 模型选型 / RAG与Agent架构设计 / Make vs Buy

## 安装与使用

### 前置条件
- 安装 [Claude Code](https://claude.ai/code)（CLI 或 IDE 扩展）

### 安装

将本仓库克隆到本地后，把 `.claude/` 目录复制到你的项目根目录：

```bash
git clone https://github.com/你的用户名/网安专家skill.git
cp -r 网安专家skill/.claude /你的目标项目/
```

或者直接在你已有的项目中引用本 Skill。

### 启动方式

**手动调用**：在 Claude Code 中输入 `/csca`，或者说「启动CSCA」「用咨询专家模式」。

**自动激活**：当你描述安全/AI咨询需求时，Claude Code 会自动匹配并激活本 Skill。触发场景包括但不限于：

- 渗透测试、安全评估、代码审计、等保、应急响应
- 安全建设方案、投标报价、培训方案
- AI安全评估、AI合规、AI战略、大模型安全
- AI赋能SOC、AI代码审计、安全Copilot

激活后会在响应开头显示 `>>> CSCA v2.1 已激活` 及当前角色。

## 工作流程

每个面向客户的交付物都强制执行四个阶段：

```
分析 → 规划 → 生成 → 审查
```

### 阶段A：分析
输出需求分析块，包括客户背景识别、核心问题定位、利益相关方分析、约束条件（技术/合规/资源/组织）、三大决策问题。信息不足时输出「信息收集清单」而不猜测。

### 阶段B：规划
输出交付规划块，包括交付物清单、方法论选择、角色声明、模板选择、风险与缓解。

### 阶段C：生成
基于模板生成交付物，严格执行去AI化写作规范。每个论断必须有依据（标准引用/行业实践/逻辑推导/经验判断/测试数据）。无法获取的客户专有数据用 `[待客户提供：XXX]` 标记，不编造。

### 阶段D：审查（两轮制）
- **第一轮**：关键点事实核查。对5-10个关键断言逐条验证（标准引用、数据一致性、计算正确性、技术事实等）。
- **第二轮**：7维度整体质量评分（专业性/逻辑性/行业适配/可实施性/完整性/可维护性/去AI程度）。任一维度低于7分自动重写，最多3轮。

## 去AI化写作

CSCA 内置严格的去AI化规则，确保输出读起来像资深顾问手写：

- **120+ 禁用词汇**：涵盖空话套话（赋能/抓手/闭环…）、AI腔连接词（值得注意的是/综上所述…）、营销腔（领先的/卓越的/一站式…）、论文公文腔（本文/笔者/旨在…）
- **表格去AI化**：7列限制、每格≤25字、禁用词检查、必须带引导句和解读句
- **句式多样性**：长短句交替、禁止连续3句相同开头、每300字至少1次节奏变化
- **证据规则**：每个结论必须满足标准引用/行业实践/逻辑推导/经验判断/测试数据至少一项

## 设计理念

1. **咨询优先**：先理解、分析、规划，再生成内容。不跳过分析直接输出。
2. **四阶段强制**：每个交付物必经完整管道，每阶段有门禁块。
3. **两轮审查**：第一轮查事实，第二轮查质量。不达标自动重写。
4. **直接交付**：输出就是最终交付件，不需要二次加工。
5. **实事求是**：不确定时明确说不知道，不编造。不夸大方案效果。
6. **合规底线**：不输出未授权的攻击自动化能力。
